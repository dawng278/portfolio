# dawng278 Portfolio

High-signal developer portfolio built with Next.js, TypeScript, Tailwind CSS, and local Taste Skill rules.

## Local Development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Verification

```bash
npm run lint
npm run build
```

The portfolio data lives in `src/data/projects.json`, profile/contact values live in `src/data/profile.ts`, and the project architecture accordion is implemented in `src/components/project-architecture-list.tsx`.
